# Clients & Transports

Clients are the core interface for interacting with Ethereum in viem-go. They handle communication with RPC endpoints and provide methods for reading and writing blockchain data.

## Client Types

viem-go provides two main client types, mirroring viem's architecture:

| Client | Description |
|--------|-------------|
| [Public Client](/docs/clients/public) | For reading blockchain data (blocks, transactions, contract reads) |
| [Wallet Client](/docs/clients/wallet) | For signing and sending transactions |

## Architecture

```
┌─────────────────┐
│     Client      │  (PublicClient / WalletClient)
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│    Transport    │  (HTTP / WebSocket)
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│   RPC Endpoint  │  (Ethereum Node)
└─────────────────┘
```

## Transports

Transports define how the client communicates with the Ethereum node:

| Transport | Description |
|-----------|-------------|
| [HTTP](/docs/clients/transports/http) | Standard JSON-RPC over HTTP(S) |
| [WebSocket](/docs/clients/transports/websocket) | Persistent connection for subscriptions |

## Basic Usage

:::code-group

```go [Go]
import (
    "github.com/ChefBingbong/viem-go/client"
    "github.com/ChefBingbong/viem-go/client/transport"
    "github.com/ChefBingbong/viem-go/chain/definitions"
)

// Create a public client
publicClient, err := client.CreatePublicClient(client.PublicClientConfig{
    Chain:     definitions.Mainnet,
    Transport: transport.HTTP("https://eth.llamarpc.com"),
})
if err != nil {
    log.Fatal(err)
}
defer publicClient.Close()
```

```typescript [TypeScript (viem)]
import { createPublicClient, http } from 'viem'
import { mainnet } from 'viem/chains'

const publicClient = createPublicClient({
  chain: mainnet,
  transport: http(),
})
```

:::

## Chain Configuration

While chain configuration is optional, it's recommended for:
- Automatic chain ID validation
- Default RPC URLs (if transport URL is omitted)
- Block time-based polling intervals

Available chain definitions:

```go
import "github.com/ChefBingbong/viem-go/chain/definitions"

definitions.Mainnet    // Ethereum Mainnet
definitions.Arbitrum   // Arbitrum One
definitions.Optimism   // OP Mainnet
definitions.Polygon    // Polygon PoS
definitions.BSC        // BNB Smart Chain
definitions.Avalanche  // Avalanche C-Chain
```
