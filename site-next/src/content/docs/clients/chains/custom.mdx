# Custom Chains

Define your own chain configurations for private networks, L2s, or any EVM-compatible chain.

## Defining a Custom Chain

```go
import "github.com/ChefBingbong/viem-go/chains"

myChain := chains.Chain{
    ID:   42069,
    Name: "My Custom Chain",
    NativeCurrency: chains.NativeCurrency{
        Name:     "Ether",
        Symbol:   "ETH",
        Decimals: 18,
    },
    RPCUrls: chains.RPCUrls{
        Default: chains.RPCUrl{
            HTTP: []string{"https://my-custom-rpc.example.com"},
        },
    },
    BlockExplorers: chains.BlockExplorers{
        Default: chains.BlockExplorer{
            Name: "My Explorer",
            URL:  "https://explorer.example.com",
        },
    },
}
```

## Using a Custom Chain

```go
c, err := client.NewPublicClient(client.PublicClientConfig{
    Chain:     myChain,
    Transport: transport.HTTP(myChain.RPCUrls.Default.HTTP[0]),
})
```

## Chain Properties

All chains share the same `Chain` struct:

| Field | Type | Description |
|-------|------|-------------|
| `ID` | `int` | Chain ID |
| `Name` | `string` | Human-readable name |
| `NativeCurrency` | `NativeCurrency` | Native token info |
| `RPCUrls` | `RPCUrls` | RPC endpoints |
| `BlockExplorers` | `BlockExplorers` | Explorer URLs |
