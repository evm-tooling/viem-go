title: getFilterChanges

Returns a list of logs or hashes based on a filter since the last time it was called.

A filter can be created from:
- [`createBlockFilter`](/docs/actions/public/blocks/create-block-filter)
- [`createContractEventFilter`](/docs/actions/public/filters/create-contract-event-filter)
- [`createEventFilter`](/docs/actions/public/filters/create-event-filter)
- [`createPendingTransactionFilter`](/docs/actions/public/filters/create-pending-transaction-filter)

### Usage

#### Event Logs

Fetch filter changes for an event filter and inspect the new logs since the last poll.

<CodeGroup
  tabs={[
    {
      title: "Go",
      language: "go",
      code: `import (
    "context"
    "log"

    "github.com/ethereum/go-ethereum/common"
    "github.com/ChefBingbong/viem-go/actions/public"
    "github.com/ChefBingbong/viem-go/client"
    "github.com/ChefBingbong/viem-go/client/transport"
    "github.com/ChefBingbong/viem-go/chain/definitions"
)

ctx := context.Background()

publicClient, err := client.CreatePublicClient(client.PublicClientConfig{
    Chain:     definitions.Mainnet,
    Transport: transport.HTTP("https://eth.llamarpc.com"),
})
if err != nil {
    log.Fatal(err)
}
defer func() { _ = publicClient.Close() }()

contractAddr := common.HexToAddress("0xa0b86991c6218b36c1d19d4a2e9eb0ce3606eb48")
filter, _ := public.CreateEventFilter(ctx, publicClient, public.CreateEventFilterParameters{
    Address: contractAddr,
})

logs, err := public.GetFilterChangesLogs(ctx, publicClient, filter.ID) // [!code focus]
if err != nil {
    log.Fatal(err)
}
log.Printf("Found %d new logs", len(logs))`,
      showLineNumbers: false
    },
    {
      title: "viem",

      code: `const filter = await publicClient.createEventFilter({

})
const logs = await publicClient.getFilterChanges({ filter })`,
      showLineNumbers: false
    }
  ]}
/>

### Returns

The return type depends on the filter type:
- `[]formatters.Log` - For event filters (via `GetFilterChangesLogs`)
- `[]common.Hash` - For block filters (via `GetFilterChangesBlocks`)
- `[]common.Hash` - For pending transaction filters (via `GetFilterChangesTransactions`)

Configuration options accepted by this action.

- **Type:** `FilterID`
