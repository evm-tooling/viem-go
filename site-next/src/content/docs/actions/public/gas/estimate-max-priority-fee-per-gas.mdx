# estimateMaxPriorityFeePerGas

Returns an estimate for the max priority fee per gas (in wei) for a transaction to be likely included in the next block.

### Returns

`*big.Int`

The max priority fee per gas in wei.

### Parameters

### Block (optional)

- **Type:** `*types.Block`

An optional pre-fetched block to use for fallback calculations. If nil, the latest block will be fetched when needed.

### JSON-RPC Method

- [`eth_maxPriorityFeePerGas`](https://ethereum.org/en/developers/docs/apis/json-rpc/#eth_maxpriorityfeepergas) (preferred)
- [`eth_getBlockByNumber`](https://ethereum.org/en/developers/docs/apis/json-rpc/#eth_getblockbynumber) + [`eth_gasPrice`](https://ethereum.org/en/developers/docs/apis/json-rpc/#eth_gasprice) (fallback)
