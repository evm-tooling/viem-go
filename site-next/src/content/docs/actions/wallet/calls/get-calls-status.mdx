# getCallsStatus

Returns the status of a call batch that was sent via [sendCalls](/docs/actions/wallet/calls/send-calls).

This implements [EIP-5792](https://eips.ethereum.org/EIPS/eip-5792).

})
if err != nil {

}
log.Printf("Status: %s (code: %d)", status.Status, status.StatusCode)`,

### Returns

`*GetCallsStatusReturnType`

An object containing:
- `StatusCode` - The status code (100=pending, 200=success, 300+=failure)
- `Status` - The status string ("pending", "success", "failure")
- `Receipts` - Array of transaction receipts
- `Atomic` - Whether the calls were executed atomically
- `ChainID` - The chain ID
- `Version` - The EIP-5792 version
- `Capabilities` - The wallet capabilities (optional)

### Parameters

### ID (required)

- **Type:** `string`

The identifier of the call batch (returned from [sendCalls](/docs/actions/wallet/calls/send-calls)).

- [`wallet_getCallsStatus`](https://eips.ethereum.org/EIPS/eip-5792#wallet_getcallsstatus) (EIP-5792)
