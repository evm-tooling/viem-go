# assertRequest

Validates a transaction request and returns an error if the request is invalid. Checks address validity and fee constraints.

## Returns

- **Type:** `error`

Returns `nil` if the request is valid, or an error describing the validation failure.

## Validations

This function validates:
1. **Account Address**: Must be a valid 40-character hex address with `0x` prefix (if provided)
2. **To Address**: Must be a valid 40-character hex address with `0x` prefix (if provided)
3. **MaxFeePerGas**: Must not exceed max uint256 value
4. **Fee Constraint**: `MaxPriorityFeePerGas` must not exceed `MaxFeePerGas` (if both are provided)
