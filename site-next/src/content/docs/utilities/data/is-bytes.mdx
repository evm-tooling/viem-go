# isBytes

Checks if a value is a byte slice (`[]byte`).

## Import

  tabs={[
    {
      title: "Go",

## Usage

<CodeGroup
  tabs={[
    {
      title: "Go",
      language: "go",
      code: `import "github.com/ChefBingbong/viem-go/utils/data"

// Valid byte slice
valid := data.IsBytes([]byte{0x01, 0x02})
// true

// Not a byte slice
valid = data.IsBytes("0x0102")
// false

// Nil is not bytes
valid = data.IsBytes(nil)
// false`,
      showLineNumbers: false
    },
    {
      title: "viem",
      language: "ts",
      code: `import { isBytes } from 'viem'

isBytes(new Uint8Array([69, 420]))
// true

isBytes('0x00000069')
// false`,
      showLineNumbers: false
    }
  ]}
/>

## Returns

- **Type:** `bool`

`true` if the value is a byte slice, `false` otherwise.

## Parameters

### value (required)

- **Type:** `any`

The value to check.

## Notes

- In Go, this is primarily useful for type assertions in generic contexts
- Returns `false` for `nil` values
- Only returns `true` for actual `[]byte` types, not other slice types
